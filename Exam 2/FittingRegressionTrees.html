Regression/Classification Trees

Tree based method:

  • Split up predictor space into regions, different predictions for each region

  • Classification tree if goal is to classify (predict) group membership
    • For a given region, usually use most prevalent class as prediction
  • Regression tree if goal is to predict a continuous response
    • For a given region, usually use mean of observations as prediction

Back to Regression Trees: How do we fit them?

  • Assuming we want to model a constant in each region, the mean is the optimal predictor

  • How do we pick our splits?

    • Fit using recursive binary splitting - a greedy algorithm

    • For every possible value of each predictor, find Residual Sum of Squares and try to minimize that

    \[R_1(j, s) = \{x|x_j < s\}\mbox{ and }R_2(j, s) = \{x|x_j \geq s\}\]

    • we seek the value of j and s that minimize the equation

\[\sum_{i:x_i\in R_1(j,s)}(y_i-\bar{y}_{R_1})^2+\sum_{i:x_i\in R_2(j,s)}(y_i-\bar{y}_{R_2})^2\]

Fitting Regression Trees

  • Once that split is chosen, the same process is used to create the second split

  • Generally grow a `large' tree (many nodes)

  • Tree then usually pruned back using cost-complexity pruning

    • Done to not overfit data

    • Prune back increases bias but decreases variance, hopefully improving prediction

  • Generally, can choose number of nodes using training/test set and comparing or using a method like cross-validation

Regression Tree Example

Obtain training and test sets

set.seed(90)
train <- sample(1:nrow(diamonds), size = nrow(diamonds)*0.8)
test <- dplyr::setdiff(1:nrow(diamonds), train)

diamondsTrain <- diamonds[train, ]
diamondsTest <- diamonds[test, ]

Regression Tree Example

treeFit <- tree(price ~ ., data = diamondsTrain)
plot(treeFit); text(treeFit)

Regression Tree Example

  • Here largest model, no pruning needed
