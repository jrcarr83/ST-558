Generalized Linear Models

  • General Linear Model (GLM):
    • Continuous response
    • Allows for both continuous and categorical predictors
  • Generalized Linear Model (GLM):
    • Allows for responses from non-normal distributions
    • Can have both types of predictors
  • Logistic Reression, Poisson Regression, Negative Binomial Regression, Multiple Linear Regression all in the GLM framework

Generalized Linear Models

  • The logit function \(\left(log\left(\frac{p}{1-p}\right)\right)\) is called the link function

  • links the mean (recall p = average number of successes at a given x) to the linear form of the model

  • Generally, we can look at a link function \(g\) where

\[g(\mu) = \beta_0 + \beta_1x_1+\beta_2x_2+...+\beta_px_p\]

where \(\mu\) is the mean for the set of \(x\) values used

Poisson Distribution

Poisson distribution is useful for modeling counts

  • Mean rate of occurrence of the distribution is usually denoted by \(\lambda\)

  • Variance is also equal to \(\lambda\) (higher mean, more variable)

  • Inference usually centered on \(\lambda\)

Poisson Regression

We'd like to model the \(\lambda_i\)

  • Issues: each ship has a different # of aggregate months, need to standardize

  • Look at modeling \(\lambda_i/exposure_i\)

  • For Poisson, use \(log\) link (natural)

\[ln(\lambda_i/exposure_i) = \beta_0 + \beta_1x_1 +... +\beta_px_p\]

  • Can now rewrite to model \(\lambda_i\) (called a log-linear model)

\[ln(\lambda_i) = ln(exposure_i) + \beta_0 + \beta_1x_1 +... +\beta_px_p\]

  • \(ln(exposure_i)\) called an offset

  • Reference is first category of type variable

Poisson Regression

Model fit:

  • One thing to look at for a Poisson regression model fit is deviance

  • Deviance is a measure of goodness-of-fit

  • A bit beyond this course, but we hope that the residual deviance/df \(\approx\) 1

  • Here much larger, overdispersion or other lack of fit present

  • Overdispersion is where the observed variance is higher than the theoretical variance
  • Theoretically for Poisson, var = mean
  • Solution: Add more explanatory variables, model overdispersion parameter, or switch to negative binomial regression

Recap!

  • Should always do Exploratory Data Analysis (EDA)
    • Plot data, summarize it across variables
  • Linear regression models
    • Use lm()
  • Generalized Linear Regresion models
    • Use glm() with appropriate link
  • Predict using predict()
