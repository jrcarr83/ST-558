Predictive Modeling

Broadly, can organize goals as

  • Unsupervised learning - goal is to learn about patterns and relationships in the data

    • No output to shoot for

  • Supervised learning - a variable or variables in the data set represent an 'output' of interest

    • May wish to make inference on the model
    • Often just want to predict response(s)

Supervised Learning

Commonly used supervised methods are

  • Regression models
    • Least squares
    • Penalized regression
    • Locally weighted/splines
    • Generalized linear models
  • Tree based methods
    • Regression/classification trees
    • Random forests, boosting, bagging
  • k Nearest Neighbors

… and many more, tons of methods!

Predictive Modeling

Linear Regression Shortcomings

Standard linear regression models

  • Assumption:
    • Response can take on any value on the real line
    • Think about an SLR model, prediction will be negative for some \(x\)
    • Often \(x\) that gives negative values not relevant so ok
  • Assumption for inference:
    • Errors normally distributed
    • Allows for t-tests and F-tests
    • Constant variance assumed (may depend on mean)

When these are violated Generalized Linear Models can sometimes help!

Generalized Linear Models

A note on names…

  • General Linear Model (GLM):
    • Continuous response
    • Allows for both continuous and categorical predictors
  • Generalized Linear Model (GLM):
    • Allows for responses from non-normal distributions
    • Can have both types of predictors
  • Confusing!

Logistic Regression

  • Since response is really success/failure –> modeling average number of successes for a given \(x\), i.e. probability of success
    • predictions should never go below 0
    • predictions should never go above 1

  • Saw modeling \(P(success|yards)\) as a linear function not great

  • Basic Logistic Regression models success probability using the logistic function

\[P(success|yards) = \frac{e^{\beta_0+\beta_1x}}{1+e^{\beta_0+\beta_1x}}\]

Logistic Regression

\[P(success|yards) = \frac{e^{\beta_0+\beta_1x}}{1+e^{\beta_0+\beta_1x}}\]

  • This function never goes below 0 and never above 1 - works great for many applications

  • The logistic regression model doesn't have a closed form solution (maximum likelihood often used to fit parameters)

Logistic Regression

  • Backsolving shows the logit or log-odds of success is linear in the parameters

\[log\left(\frac{P(success|yards)}{1-P(success|yards)}\right) = \beta_0+\beta_1 yards\]

  • Coefficient interpretation changes greatly from linear regression model!

  • \(\beta_1\) represents a change in the log-odds of success

  • Great way to think about modeling!
