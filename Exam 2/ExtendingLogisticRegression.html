Generalized Linear Models

  • The logit function \(\left(log\left(\frac{p}{1-p}\right)\right)\) is called the link function

  • links the mean (recall p = average number of successes at a given x) to the linear form of the model

  • Generally, we can look at a link function \(g\) where

\[g(\mu) = \beta_0 + \beta_1x_1+\beta_2x_2+...+\beta_px_p\]

where \(\mu\) is the mean for the set of \(x\) values used

  • For standard linear regression \(g(\mu) = \mu\) (identity link)

    • Recall value predicted is mean response at a given x
