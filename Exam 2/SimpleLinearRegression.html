Predictive Modeling

Broadly, can organize goals as

  • Unsupervised learning - goal is to learn about patterns and relationships in the data

    • No output to shoot for

  • Supervised learning - a variable or variables in the data set represent an 'output' of interest

    • May wish to make inference on the model
    • Often just want to predict response(s)

Unsupervised Learning

Commonly used unsupervised methods are

  • Cluster analysis

  • Market basket analysis

  • Multiple dimensional scaling (MDS)

  • Principal Components Analysis (PCA)

  • Factor analysis

An inherently more difficult problem than supervised learning!

Supervised Learning

Commonly used supervised methods are

  • Regression models
    • Least squares
    • Penalized regression
    • Locally weighted/splines
    • Generalized linear models
  • Tree based methods
    • Regression/classification trees
    • Random forests, boosting, bagging
  • k Nearest Neighbors

… and many more, tons of methods!

Linear Regression Ideas

Recall the basic SLR model:

\[Y_i = \beta_0 + \beta_1x_i + E_i\] Here - \(Y_i\) is our response for the \(i^{th}\) observation
- \(x_i\) is the value of our explanatory variable for the \(i^{th}\) observation
- \(\beta_0\) is the y intercept
- \(\beta_1\) is the slope

Model fit by minimizing the sum of squared residuals
\[min_{\beta_0,\beta_1}\sum_{i=1}^{n}\left(y_i-\beta_0-\beta_1x_i\right)^2\]

Linear Regression ideas

  • Minimizing sum of squared errors is equivalent to assuming normality on errors and using maximum likelihood to estimate the \(beta\)'s

Basic Analysis

  • Add PI easily with packages (install ciTools package)
cars <- cars %>% ciTools::add_pi(fit, names = c("lower", "upper"))

ggplot(cars, aes(x = speed, y = dist)) +
  geom_point() + 
    geom_smooth(method = "lm", fill = "Blue") +
    geom_ribbon(aes(ymin = lower, ymax = upper), alpha = 0.3, fill = "Red") +
    ggtitle("Scatter Plot with 95% PI & 95% CI")
